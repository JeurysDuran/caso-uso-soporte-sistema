# Caso de Uso: Recuperación de Acceso al Sistema

## Actor
Usuario

## Descripción
El usuario no puede acceder al sistema debido a contraseña olvidada o bloqueo de cuenta.

## Flujo Principal
1. El usuario reporta el problema al área de soporte.
2. Soporte valida la identidad del usuario.
3. Se restablece la contraseña o se desbloquea la cuenta.
4. El usuario accede nuevamente al sistema.

## Resultado Esperado
El usuario recupera el acceso exitosamente.
